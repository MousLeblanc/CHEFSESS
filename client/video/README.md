# Vidéo de la Landing Page

## Instructions

Placez votre vidéo dans ce dossier avec le nom : `collective-catering.mp4`

## Recommandations

- **Format** : MP4
- **Résolution** : 1080p (1920x1080) minimum
- **Orientation** : Horizontale (landscape)
- **Durée** : 30-60 secondes (en boucle)
- **Poids** : Optimisé pour le web (< 10 MB recommandé)

La vidéo doit montrer :
- Cuisiniers dans une cuisine collective
- Service de repas en collectivité (école, hôpital, EHPAD)
- Ambiance professionnelle et chaleureuse
- Mouvement et dynamisme
